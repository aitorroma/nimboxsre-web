# NimBox SRE web

Landing estática para **NimBox SRE — Operaciones y observabilidad gestionada**.

## Desarrollo

Abre `index.html` directamente en un navegador o sírvelo con cualquier servidor estático:

```bash
python3 -m http.server 8000
```

El diseño toma como referencia el enfoque editorial B2B de la web de Invoke, pero el contenido, estructura y estilos son propios.
